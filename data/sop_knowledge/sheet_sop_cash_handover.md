# SOPs — Cash Handover

## COD pendency reflecting in the system

**Category:** Cash Handover
**TAT:** 
**Keywords:** cod|deposit|pendency

### Scenario
L1 agent will refer to this query to check the latest COD pendency of the hub

### L1 Process (What agent must do)
L1 agent will refer to this query to check the latest COD pendency of the hub Next, L1 agent will use this query to check the transaction details by entering either the hub code/Transaction ID If there is no COD pendency showing up, then agent can respond to the ticket using the following template Template: Dear Partner, The current COD pendency for your hub is reflecting as 0. The last deposit from your hub was received on [date] for an amount of Rs. XX. We will be closing the ticket for now. You may reopen the ticket after 48 hours if the issue persists. For any further assistance, feel free to reach out. Thanks you, Valmo Partner Support.
