### Scenario
Scenario 3 - Captain wants to track their order / Haven’t received their order on time / wants to know the status of their order:

### L1 Process (What agent must do)
Before solving for the Scenarios the agent must check if the captain has provided Order ID and Hub ID to simplify the verification process. If any of the information required per the scenarios is unavailable then reply to the partner following this template. Agent should follow the instructions when faced with the following scenarios: Agent must look up the order ID mentioned by the partner on the RVP consumables partner details by applying a filter or using Ctrl+F. Agent must view the corresponding values in the columns - Supplier Code, Dispatch Date, Delivery Date, Order Status, Docket no(which is the tracking id) and the courier name. If the Order Status = Delivered then loop in the Supplier using the email linked to the supplier code in this sheet. (Discussion Pending with Cons team) If order status = dispatched, agent must visit the tracking site of the Partner mentioned in the courier name column and enter the value in the docket no column to track the parcel. (For example courier name = Amazon, type “Amazon parcel tracking” on the search engine and enter the tracking ID). Reply to the partner by mail mentioning the order status in the website and telling them how to track it themselves following this template.

### Scenario
Scenario 4 - Captain has received lesser consumables than they have ordered:

### L1 Process (What agent must do)
Before solving for the Scenarios the agent must check if the captain has provided Order ID and Hub ID to simplify the verification process. If any of the information required per the scenarios is unavailable then reply to the partner following this template. Agent should follow the instructions when faced with the following scenarios: Agent must look up the order ID mentioned by the partner on the RVP consumables partner details by applying a filter or using Ctrl+F. Agent must view the following columns corresponding to the order ID - Order details, qty and order status. Agent must cross verify the quantity mentioned by the Partner and the quantity mentioned in the RVP consumables partner details sheet. (order Details amount x qty) If the quantity is matching and the status = delivered then reply to the partner using the following template and loop in the Supplier using the email linked to the supplier code in this sheet. If the quantity is matching and the status is dispatched, then reply to the partner using the following template. If the quantity is not matching then loop in the Supplier using the email linked to the supplier code in this sheet.
