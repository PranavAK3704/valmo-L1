# Bag ID to AWB Extraction via Valmo 2k24 Lost Query

BAG ID TO AWB EXTRACTION: When a ticket contains Bag IDs (format: UN followed by digits, e.g. UN117197848) but NO individual AWB numbers, do NOT ask the captain for AWBs. Instead, enter the Bag ID into the Valmo 2k24 lost Metabase query to extract the individual AWB numbers. Bag IDs are container IDs holding multiple AWBs. Always look up AWBs from Bag IDs before using needs_info to request them from captain.
